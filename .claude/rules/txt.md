# テキストファイルルール

## hikitsugi.txt（引き継ぎメモ）

このファイルは作業の引き継ぎ記録として使う最重要ファイル。

- 作業開始時は必ず最初に読む。
- 作業終了時は日付・作業内容・次のステップを追記する。
- モモさんは `続き.bat`（Windows）または `続き.ps1`（PowerShell）で開く。

### 記載フォーマット

```
# ひきつぎメモ（YYYY-MM-DD 更新）

## 作業まとめ

### やったこと
- ...

### 次にやること
- ...
```

## その他の .txt ファイル

- `00_ももさん用/` 内のファイルはモモさん向け操作ガイド。内容の書き換えは最小限にする。
- 文字コードは UTF-8 で統一する。
