# 思春期対応作戦会議 ── プロジェクト指示書

## 📋 プロジェクト概要

モモさん（思春期子育て専門保健師・桃木亜子）の、**発信 → 講座 → Kindle本の統合コンテンツ制作システム**

### ビジネス全体構造

```
発信軸（日々の学び・気づき）
├─ Threads（愛のある喝・背中押し）
├─ Note（Q&A形式・記事化）
├─ stand.fm（音声配信）
└─ Kindle本（「子どもが部屋から出てこない時の3STEP」）
        ↓
    メルマガ（1本・子どもが不調の時のお母さんのマインドセット）
        ↓
    講座選択（入り口は複数）
    ├─ セルフケア講座（全員向け基礎）
    └─ 子どもサポート講座（選択的・詳細学習）
```

## 🎯 講座の3つの柱（すべてのコンテンツの軸）

### ① 親自身が安定していることが大切
**メッセージ**：「お父さん、お母さん、あなたは大事にされるべき人なんですよ」

- 親は子どもや家族を優先しがち → 自分自身を後回しにしている
- 本来、親こそが大事にされるべき人。人は誰でも幸せになる権利がある
- 満たされない思いも、誰にも満たしてもらえない時も、自分自身は満たすことができる
- **具体的スキル**：セルフケアのエビデンスのあるテクニック（マインドフルネス、セルフコンパッション、など）
- **実現の場**：セルフケア講座・発信全般

### ② 思春期の子どもの脳と心は「別物」
**メッセージ**：「今までの対応が通用しないのは、お母さんのせいではなく、子どもの脳の構造が変わったから」

- 思春期の脳と心と身体は、思春期前とは別物になっていく
- 子どもの発達と思春期の発達目標を理解することが、関わり方を変える
- 思春期の子育ての目標：親がいなくなっても生きていける「自立」を育てる
- 親がこれを知ると、子どもとの関係がよくなり、子どもは困った時に親に頼ることができる
- **実現の場**：セルフケア講座（発達理解）・子どもサポート講座（実践的対応）

### ③ 見守るだけでなく、具体的に動く
**メッセージ**：「『見守る』を『将来を見据えて、動く』に変える」

- 子どもが部屋から出てこない時：「見守る」で止まっていないか？
- 具体的には：いつまで？　何をする？　どういう道筋を立てる？
- ただ見守るのではなく、「次に何をすべきか」「どこに相談すべきか」を示す
- 見守る不安を、具体的な行動計画に変えていく
- **実現の場**：子どもサポート講座・Kindle本・発信

## 📢 発信軸（Threads・Note・stand.fm）

### 全体フロー：日々の学び → Threads → Note → スタエフ → 講座

#### Threads
- **スタイル**：「愛のある喝」（共感 → 背中押し → 希望）
- **配信ルール**：3パターン常時作成
- **パターン①②③**：基本（個別ポストで完結）
- **パターン④**：ツリー型（1投稿目で興味を引く → コメント欄で3～4枚で解説）

#### Note
- **形式**：Q&A形式（冒頭 → セクション①～⑥ → 末尾テンプレート）
- **テンプレート**：別途ファイル参照
- **用途**：Threads や深掘り項目 120 から発展させた記事化

#### stand.fm（スタエフ）
- **配信形式**：音声配信＋概要欄（データリンク・Note・メルマガ・Kindle本等へのリンク含む）
- **固定テンプレート**：
  - オープニング：「思春期子育て専門保健師の桃木亜子です」
  - エンディング：「焦らなくて大丈夫。あなたが聴いてくれているだけで、もう十分です」
- **台本**：Google ドキュメントで管理

#### Kindle本
- **企画フェーズ**：インタビュー募集（Threads で告知）
- **執筆フェーズ**：各章の素材を発信コンテンツから抽出
- **公開後**：メルマガ・発信で誘導

## 作業ルール

### 保存先
- メルマガ原稿：`C:\Users\Owner\Documents\claude_workspace\思春期対応作戦会議\01_発信軸\メルマガ`

### メルマガ保存フロー
メルマガを貼り付けて「保存してください」と言われたら、以下の2ステップを必ず両方実行する。

**① メルマガ原稿を保存**
- 保存先：`01_発信軸/メルマガ/`
- ファイル形式：`.txt`
- ファイル名：`メルマガ_テーマ名.txt`（シリーズの場合は `メルマガ_○○シリーズ全X回.txt`）

**② 関連講座フォルダに参考素材として保存**
- メルマガの内容を読み、関連する講座（セルフケア講座の何回目 or 子どもサポート講座）を判断する
- 保存先：`02_講座/self_care/素材・アイデア帳/` または `02_講座/child_support/素材・アイデア帳/`
- ファイル形式：`.md`
- ファイル名：
  - セルフケアの場合：`セルフケア講座_第X回テーマ_参考素材_ファイル名.md`
  - 子どもサポートの場合：`子どもサポート講座_参考素材_テーマ名.md`
- 内容構成：出典 → 主要テーマ → 人物エピソード／具体例 → 講座に組み込むポイント
