```
罪悪感の悪循環

子どもが自分を責める
→ お母さんも自分を責める
→ 脳が危険モード（逃走反応）
→ 子どもがお母さんに攻撃する
→ お母さんが子どもを追い詰める
→ 悪循環が回り続ける

エネルギー曲線
頑張って学校に行く → 限界で一気に行けなくなる
→ 底が続く → 少しずつ回復する
→ この回復を邪魔しないことが大事

二次障害
罪悪感が膨らむと二次障害につながる
追い詰めないことが最優先

## トークスクリプト（全枚分）

```
【受講するにあたってのお願い】
今日は1時間、一緒に過ごしましょう。
お子さんやペットが入ってきても全然大丈夫です。
お飲み物も自由にどうぞ。

録画とスライドは講座後にPDFでお送りします。
なのでメモを取らなくても大丈夫です。
今日は聴くことだけに集中してください。

質問や「うちはちょっと違う」という意見も
いつでも歓迎です。
むしろ、そういう声が場を豊かにしてくれます。


【今日のゴール】
動画では、罪悪感の悪循環を断ち切ることが
一番大事だとお伝えしました。

今日はその先、具体的にどう動けばいいかを
一緒に考えていきたいと思います。

「今日来て良かった」と思って帰ってもらえるように
精一杯お伝えします。


【リラックスして聴いて欲しい】
その前に、まず体を整えましょう。

楽な姿勢をとってください。
背筋を伸ばさなくて大丈夫です。

1・2・3で鼻から息を吸いながら、お腹を膨らませます。
4で一旦止める。
5・6・7・8・9・10でゆっくり息を吐きながら、お腹を凹ませます。

私はやるときにラベンダーやユーカリのオイルを嗅ぎながらやっています。
香りが脳のモヤモヤを全部巻き込んで、吐き出すイメージです。

これを4〜5回繰り返しましょう。
一緒にやってみてください。


【今日の内容】
今日は3つのことをお話しします。

まず、学校を休んだ子どもにどんな経過をたどるか。
次に、回復の鍵になる3つの視点。
そして、今日から使える具体的な方法です。

盛りだくさんですが、一つひとつ丁寧にお伝えします。


【自己紹介のお願い】
まず、皆さんのことを聞かせてください。

お子さんのこと、今どんな状況か。
それからお母さん自身の今の状態を
教えていただけますか？

（一人ずつ聞いていく）

皆さんそれぞれ状況が違いますね。
今日はそれぞれの状況を頭に置きながら
お話ししていきます。


【不登校はお母さんのせいじゃない】
まず最初に、これだけはっきり言わせてください。

不登校はお母さんのせいじゃありません。

でも、関わり方によって
子どもの回復が早くなることはあります。

これは責めているのではなく、
お母さんにできることがあるという希望の話です。
今日はその話をしていきます。


【罪悪感の悪循環】
動画でもお伝えしましたが、まず全体像を確認しましょう。

子どもが自分を責める。
するとお母さんも自分を責める。
そうすると脳が危険モード、逃走反応に入ります。
子どもはそのお母さんの状態を敏感に感じ取って、
攻撃してくることがあります。
するとお母さんはさらに追い詰められる。
この悪循環がぐるぐると回り続けます。

エネルギーの曲線で見ると、
子どもは頑張って学校に行く、でも限界で一気に行けなくなる。
そこから底が続いて、少しずつ回復していきます。
この回復の時期を邪魔しないことが、一番大事です。

罪悪感が膨らむと二次障害につながることがあります。
だから、追い詰めないことが最優先なんです。


【回復力を上げる3つの視点　アジェンダ】
では、どうすれば回復の邪魔をせずに
子どもをサポートできるか。

3つの視点でお話しします。

①赤い脳を冷やす
②罪悪感・劣等感の悪循環を断つ
③お母さんの心を安定させる

この3つです。


【① 赤い脳を冷やす】
動画でお伝えした「赤い脳」、覚えていますか？

逃走反応が続いている状態、いわゆる赤い脳の状態では
子どもは何も考えられません。
学校に行くかどうかより先に、
まずこの赤い脳を冷やすことが必要です。

そのために一番大事なのは2つ。

安心・安全を感じられるお家にすること。
そして、罪悪感を感じにくくする関わりをすること。

この2つが、すべての土台になります。


【② 罪悪感・劣等感の悪循環を断つ】
子どもは、誰よりも自分を責めています。

「なんで自分は行けないんだろう」
「みんなは行ってるのに」
そう思いながら、毎日を過ごしています。

だから大事なのは、
「今は休むことが必要な時期なんだ」と
子どもが感じられる環境をつくること。

そしてお母さんが落ち着いていること。
それだけで、子どもの罪悪感はずいぶん和らぎます。

お母さんの状態が、子どもに直接影響しているんです。


【③ お母さんの心を安定させる】
3つ目は、お母さん自身のことです。

自分を責めるのをやめてください。
一人で抱えないでください。
誰かとつながってください。

お母さんが整うことが、子どもの回復につながります。

これは犠牲になれということではありません。
お母さん自身が安定することが、
一番の子どもへのサポートになるということです。


【今日から使える具体的な方法　アジェンダ】
では具体的に何をすればいいか。

4つお伝えします。

①自主休校式
②学校とのつながり方
③友達への伝え方
④新たな対応をするときのコツ

一つひとつ見ていきましょう。


【出典紹介】
ここからお伝えする「自主休校式」は、
井上ゆうき先生の
「学校では教えてくれない自分を休ませる方法」
という本をベースに、私がアレンジしたものです。

とても良い本なので、興味があればぜひ読んでみてください。


【自主休校式の導入】
「本当は行くべきなのに、休んでいる」

この感覚が、子どもの罪悪感を一番大きくします。

これを「今は休むことが必要な時期なんだ」という
雰囲気に変えていく。
それが自主休校式の考え方です。

ただ様子を見るだけじゃなく、
休んでいる間の決まりごとを家族で共有する。
これが大きな違いを生みます。


【回復の４段階】
子どもの回復には段階があります。

0段階は、子どもとつながれていない状態。
1段階は、安全確保。子どもが安全な場所で過ごせていて、家族がそれを認められている状態。
2段階は、生活リズムを整える。その子のペースで。
3段階は、好きなことをする。
4段階は、未来に向かって動き始める。

今お子さんはどの段階にいますか？
段階によって、やることが違います。
焦って先の段階に引っ張ろうとしないことが大切です。


【学校とのつながり方】
学校とどうつながるか、これも子どもと一緒に決めます。

「どこまで先生に話すか」
「先生に会いに行くか」
「Zoomで授業に参加できるか聞いてみるか」

これを親が勝手に決めるのではなく、
子どもと相談して決める。

それだけで、子どもは「自分のことを自分で決めている」
という感覚を持てます。
これが回復にとってとても大事です。


【受講生の実話】
実際にこんなことがありました。

小学校高学年のお子さんのケースです。
学校と情報共有しながら、よい関係を続けていました。

校外学習で山登りがあった日、
担任の先生がLINEビデオ通話をしてくれたんです。
クラスメイトが山を登っている様子を
家にいながら一緒に見ることができた。
家で応援することができた。

学校に行けなくても、こういうつながり方もあります。
学校との関係を切らずにいることで、
こういう温かいことが起きることもあります。


【友達への伝え方】
友達への伝え方も、子どもと一緒に決めます。

本音を言える仲の良い友達には何を話すか。
クラスの一般的な友達にはどう答えるか。

LINEで「最近どうしてるの？」と聞かれたとき、
何て返すか。これを一緒に考えておくだけで、
子どものストレスがぐっと減ります。

子どもから「どう答えればいい？」と聞いてきたときが
一緒に考えるチャンスです。


【新たな対応をするときのコツ】
今日学んだことを家で試すとき、一つコツがあります。

「今日保健師さんと相談したんだよ」と子どもに伝えること。

そして「こうしていこうと思うんだけど、
あなたはどう感じる？」と聞くこと。

これだけで、子どもは「自分も関わっている」と感じます。
親が一方的にルールを変えるのではなく、
一緒に決めていく形にする。
それが、やりやすくなる一番のコツです。


【今日のまとめ】
今日お伝えしたことを3つにまとめます。

1つ目。子どもの回復力を上げる関わり方があります。
闇雲に見守るのではなく、段階に合った関わりをすること。

2つ目。お母さんが整うことが、子どもの回復につながります。
お母さん自身を大切にしてください。

3つ目。子どもと一緒に決めて、一緒に進む。
子どもを置いてけぼりにしないこと。

この3つです。


【おわりにメッセージ】
最終的なゴールは「自立」です。

学校に戻ることではなく、
親の手を離れて、自分の力で生きていけること。

それが本当のゴールです。

そのために今、何が必要か。
一緒に考えていきましょう。


【今考えるべきこと】
今考えるべきは、学校に行くかどうかではありません。

今この子が回復するために、何が必要か。

この子にどう対応するか。
家族の理解をどう得るか。
学校にどう対応するか。

状況は一人ひとり違います。
その状況を見極めながら、今何をすればいいか。
それを一つひとつ、焦らず考えていきましょう。


【今日聞いてできそうなこと】
今日聞いた中で、一つだけでいいです。

「これならできそう」というものはありましたか？

完璧にやろうとしなくていいです。
一つだけ、明日から試してみてください。
