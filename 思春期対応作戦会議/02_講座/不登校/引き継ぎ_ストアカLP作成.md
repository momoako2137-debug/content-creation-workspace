# 引き継ぎ：ストアカ連動LP 新規作成（2026-06-09）

## やること
ストアカの講座ページから来た人向けの、シンプルな動画ページをWordPressに作る。

---

## ページの位置づけ
- ストアカの講座ページ → このWPページへリンク → 動画を見てもらう → 個別相談・OKサロンへ誘導
- ストアカページに共感文・レビューが全部あるので、このページはシンプルでよい

## ページ構成
1. キャッチコピー
2. 動画
3. 個別相談CTA
4. OKサロンCTA
5. プロフィール（簡潔）

## 含めないもの
- こんな状況ではありませんか（ストアカページにある）
- 個性発見大学・ストアカの説明
- 個性発見大学の紹介文

---

## スクリプト
- ファイル：`02_講座/不登校/wp_stoaka_lp.py`
- 実行コマンド（下書き作成）：
  ```
  cd 思春期対応作戦会議\02_講座\不登校
  python wp_stoaka_lp.py
  ```
- 実行コマンド（公開）：
  ```
  python wp_stoaka_lp.py publish
  ```
- 実行コマンド（既存ページ更新）：
  ```
  python wp_stoaka_lp.py update
  ※ EXISTING_PAGE_ID に作成後のページIDを記入してから実行
  ```

---

## 作業手順
1. `python wp_stoaka_lp.py` を実行して下書き作成
2. 表示された編集URLをブラウザで開いて確認
3. 問題なければ `python wp_stoaka_lp.py publish` で公開
4. ストアカ講座ページの説明文にこのWPページのURLを追記してもらう（モモさん作業）
