# 子どもサポート講座 全13回 ── 全体構成と3つの柱

## 講座の位置づけ

| 項目 | 説明 |
|---|---|
| **対象** | 子どものメンタル不調に直面している親 |
| **前提** | セルフケア講座（①親の安定）を学んだ後の詳細学習 |
| **メイン柱** | ②子どもを理解する + ③具体的に動く |
| **全体テーマ** | 「見守るだけでなく、家の内側と外側を整え、親子で一緒に歩む」 |

---

## 全13回の構成と3つの柱での分類

### **第0回：オリエンテーション（配信週0）**

**メイン柱**：①②（親のマインドセット + 子どもを理解する入口）

#### セクション
- 01 はじめに〜大丈夫、必ずよくなる
- 02 あなたのせいじゃない
- 03 良かれと思って憎み合っていないか
- 04 手放していい思い込み
- 05 この講座のゴール
- 06 この講座で学ぶこと・全体の地図
- 07 受講の進め方・サポートの使い方

#### ねらい
- 親の罪悪感・不安を軽くする
- 講座全体の全体像を理解
- 「大丈夫」という希望を植え付ける

#### セルフケア講座からの関連
→ セルフケア講座 第0回・第1回・第10回の学びを、このコンテキストに落とし込む

## セルフケア講座との関連性

| 講座 | 役割 | メイン柱 |
|---|---|---|
| **セルフケア講座** | 親の土台づくり | ①親の安定 |
| **子どもサポート講座** | 親の実践スキル | ②子どもを理解 + ③具体的に動く |

```
セルフケア講座（13回）で親が安定する
    ↓
子どもサポート講座（13回）で親が「動く力」を身につける
    ↓
親子で一緒に歩む
```
